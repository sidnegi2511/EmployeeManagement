# Employee Management System

## Overview

The Employee Management System is a web application that provides CRUD (Create, Read, Update, Delete) operations for managing employees. It is built using Angular 18 for the front-end and .NET Core 7 with Entity Framework for the back-end. For the database, an InMemoryDB is used. The back-end API is documented and tested using Swagger.

## Features

- Add new employees
- View a list of employees
- Update employee details
- Delete employees
- InMemoryDB for database operations
- Swagger integration for API testing and documentation

## Technologies Used

### Front-End
- Angular 18

### Back-End
- .NET Core 7
- Entity Framework Core
- InMemoryDB for database

### API Testing & Documentation
- Swagger
